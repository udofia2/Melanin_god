## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
## Run Database commands

```bash
# database migration 
$ yarn prisma:migrate

# generate
$ yarn prisma:generate

# reset database
$ yarn db:reset

# start prisma studio
$ yarn prisma:studio
```
